What is Avalanche Bridge?

Avalanche Bridge is the official bridging experience surfaced through Core for moving supported assets between Ethereum and Avalanche. See the Avalanche Bridge support docs for current guidance.

How Avalanche Bridge works

  1. Open CoreUse the official Core bridge page and connect the wallet that holds the asset you want to move.
  2. Choose networksSelect Ethereum or Avalanche as the source, then pick the destination network and supported token.
  3. Review detailsCheck the displayed route, estimated fees, destination address, and any token-specific notes before signing.
  4. Confirm transferApprove the wallet transaction and track completion from Core or the relevant network explorer.

Before you bridge

Built for Core users

The bridge flow lives inside Core, Ava Labs' wallet app for managing, sending, swapping, and bridging crypto.

Network-aware routing

Transfers are shown with source and destination networks so users can verify they are using Ethereum and Avalanche C-Chain correctly.

Support-backed guidance

Avalanche maintains bridge help articles covering usage, fees, security, USDC behavior, and troubleshooting in its support center.

Avalanche Bridge FAQ

What is avalanche bridge used for?

It is used to move supported assets between Ethereum and Avalanche, with the current app flow handled through Core.

Is avalanche bridge the same as Core?

Avalanche Bridge is the bridge product; Core is the official app interface where users access the bridge flow.

What fees apply to avalanche bridge?

Fees vary by direction, asset, and network conditions; review the live quote in Core and the official fee article before confirming.

Can avalanche bridge send to a different wallet address?

For Ethereum-Avalanche bridge transfers, official support notes that transfers are to the same address on the other network; send onward after the bridge completes if needed.

Where is the official avalanche bridge app?

The official bridge app route is core.app/bridge. Always verify the domain before connecting a wallet.
